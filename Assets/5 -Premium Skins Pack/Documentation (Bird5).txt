	
--------------Bird #5(Pigeons & Sparrows)---------------------------

This package contains 2D animated Cute Birds, that will perfectly fit in a side scrolling platformer games and any other type of games.All animations are Sprite-sheet animations.You can use the package commercially and non-commercially

Technical Details:-
		
		- Character's : 6 (Pigeons x 3, Sparrows x 3)
		- Props : 8(Chilli x2, Tomatto x2, Carrot, Gold coin x2, Bird cage.etc)
		- Animations : 1 (Flying)Spritesheet animation
		- Design type : Cartoony, 2D
		- File format : PNG
		- Sprite resolution :  4685 x 3336 (Parrot:individual), 2016 x1910 (Props : Packed image)
			
			
Features:-
	- Different unique Birds
	- Cute and funny character
	- Smooth loop animations
	- Easy to adjust speed of the animation's using unity editor
	- Prefabs included
	- Rigidbody, 2D Box colliders attached
	- Drag and drop and use
	- Well arranged folders
	- High quality images
	- Mobile supported
	- Demo scenes & Guides included
	
How to use:-
	
	Simply drag and drop any prefab from(Birds_5_Pigeons > Prefabs) folder
	to your scene and use it.
